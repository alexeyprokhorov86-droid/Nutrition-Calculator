# Калькулятор БЖУ - Кондитерская Прохорова

Веб-приложение для расчета пищевой ценности (белки, жиры, углеводы, калории) на 100г готовой продукции.

## Как запустить локально

1. Установите Python 3.8 или выше
2. Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

3. Запустите приложение:
```bash
streamlit run nutrition_calculator.py
```

4. Откройте браузер по адресу: http://localhost:8501

## Как использовать

1. **Поиск ингредиента**: Введите название (например: "мука", "сахар", "масло")
2. **Выбор из списка**: Выберите нужный ингредиент из найденных
3. **Укажите количество**: Введите количество в граммах
4. **Добавьте в рецепт**: Нажмите кнопку "Добавить в рецепт"
5. **Просмотр результатов**: Справа автоматически отобразятся БЖУ и калории на 100г

## Как развернуть на Streamlit Community Cloud

1. Создайте аккаунт на GitHub (github.com)
2. Создайте новый репозиторий
3. Загрузите туда файлы:
   - nutrition_calculator.py
   - requirements.txt
   - ingredients.csv

4. Зайдите на streamlit.io/cloud
5. Подключите свой GitHub репозиторий
6. Выберите файл nutrition_calculator.py как главный
7. Нажмите Deploy

## Структура файлов

- `nutrition_calculator.py` - основной код приложения
- `requirements.txt` - список необходимых библиотек
- `ingredients.csv` - база данных ингредиентов
- `README.md` - этот файл с инструкциями
